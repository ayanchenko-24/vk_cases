Проект: Парсинг бизнес-кейсов с сайта VK Ads
Этот небольшой учебный проект я сделала для задания по теме «парсинг HTML-страниц с помощью BeautifulSoup».
Задача проекта — извлечь информацию о бизнес-кейсах с сайта https://ads.vk.com/cases, но парсинг выполняется не с интернета, а с заранее скачанного HTML-файла cases.html.

Что делает программа
Открывает сохранённую HTML-страницу.
Находит все карточки кейсов (виджет-карточки).
Извлекает из каждой карточки:
название кейса;
ссылку на страницу кейса (если ссылка относительная — преобразует в абсолютную);
дату публикации (если есть).
Приводит дату к общему формату YYYY-MM-DD.
Сохраняет результат в файл results.json в виде списка словарей.
Используемые технологии
Python 3
BeautifulSoup4 — для парсинга HTML
lxml — парсер
dateparser — для преобразования русских дат
json — сохранение результата
Структура проекта
project-folder/ │ ├── src/ │ └── parse_cases.py # основной скрипт парсинга │ ├── cases.html # сохранённая HTML-страница сайта VK Ads ├── results.json # файл с результатами (создаётся автоматически) │ ├── README.md # описание проекта ├── requirements.txt # список зависимостей └── .gitignore # исключения

Как запустить проект (пошагово)
Клонировать репозиторий
git clone <ссылка_на_мой_репозиторий>
cd vk-cases-parser

2. Создать виртуальное окружение
python -m venv .venv

3. Активировать окружение (macOS / Linux)
source .venv/bin/activate

4. Установить зависимости
pip install -r requirements.txt

5. Поместить файл cases.html в корень проекта
Файл нужно сохранить из браузера

Запуск программы
cases_parser.py

После запуска появится файл:
results.json
В нём будут собраны данные всех найденных кейсов.
